---

##### 🧐 다음 출력 값으로 옳은 것은?

```js
var arr = [100, 200, 300];
console.log(typeof arr);
```

1.  undefined
2.  string
3.  number
4.  object

---

##### 🙋🏻‍♀️ 답안

4. object

---

#### 🧩 typeof

`typeof`는 데이터 형식을 확인하는 연산자이다.

- undefined: 정의되지 않거나 값이 없는 경우
- boolean: 논리 값 (true ∙ false)
- number: 숫자
- string: 문자열
- object: 객체 ∙ null (객체는 null을 포함!)
- function: 함수
- symbol: ES6에서 추가된 Symbol 형식
